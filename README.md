# TeachAthon 2025

Welcome to the official public archive for **TeachAthon 2025**, hosted by the **Synthetic Processes & Embodiment Labs (SPEL)**.

This repository documents the seven shortlisted projects developed during the event, including their project profiles, team details, demonstrations, and original submissions.

---

## 🏆 Official Placements and Project Guide

Select a project name below to open its dedicated folder, where you can read a concise project overview, view the team details, access available demonstrations, and download the original submission document.

| Placement                    | Project / Application                         | Team Members                                                                                      | Project Focus                                                                                                                                    |
| :--------------------------- | :-------------------------------------------- | :------------------------------------------------------------------------------------------------ | :----------------------------------------------------------------------------------------------------------------------------------------------- |
| 🥇 **Winner — 1st Place**    | [LeQture](./Submissions/LeQture/)             | Shlok Gupta                                                                                       | Context-aware AI support embedded into recorded lectures, with in-video questions, citations, quizzes, flashcards, and revision materials.       |
| 🥈 **Runner-up — 2nd Place** | [StudentVerse](./Submissions/StudentVerse/)   | Poorna Siri Karanam                                                                               | Gamified AI planning that turns academic and personal goals into daily micro-tasks and rewards progress through an evolving virtual universe.    |
| 🥉 **3rd Place**             | [GradeIQ](./Submissions/GradeIQ/)             | Matvii Yatsenko, Viktor Sharha                                                                    | AI-assisted marking of typed and handwritten PDF submissions using editable rubrics, criterion-level feedback, and human review.                 |
| **4th Place**                | [StudyAI](./Submissions/StudyAI/)             | Caroline Rennier, Orkun Kınay                                                                     | Course-grounded Socratic AI tutoring that adapts to students, tracks misconceptions, and verifies its explanations across multiple AI modules.   |
| **5th Place**                | [MindfuLLM](./Submissions/MindfuLLM/)         | Jason Latz, Benedek Labancz, Ivan Sowerby, Dhyey Mehta, Alp Niksarli, Ömer Tuğrul, Barış Tekdemir | A browser extension that converts selected academic LLM conversations into personalised multiple-choice and free-response quizzes.               |

## 📂 Repository Structure and Navigation

Project materials are organised into separate folders inside the **`Submissions/`** directory:

```text
Submissions/
├── LeQture/
├── StudentVerse/
├── GradeIQ/
├── StudyAI/
├── MindfuLLM/
├── FormativED/
└── Symbol-Sense/
```

Each project folder contains:

* **`README.md`** — the project's profile page, including its purpose, principal features, team members.
* **Original submission document** — the project report submitted during TeachAthon 2025.
