## 📂 Repository Structure & How to Navigate

If you are browsing the raw files directly, the repository is organized cleanly by project topic to ensure long-term academic utility:

* **`Submissions/`**: Contains separate folders for each shortlisted app.
  * `[AppName]/README.md`: The team's profile card containing project details and developer credits.
  * `[AppName]/[Document].docx`: The original, curated Microsoft Word submission file available for download.
